# 🛒 Kiosco-CRM

Sistema de gestión integral para kioscos y pequeños comercios.  
Desarrollado en Python con arquitectura modular, permite administrar ventas, inventario, caja, proveedores y usuarios en un solo sistema.

---

## 🚀 Características

- 🛍️ Punto de Venta (POS) intuitivo  
- 📦 Control de inventario en tiempo real  
- 💰 Gestión de caja (apertura, cierre, movimientos)  
- 👥 Administración de usuarios y roles  
- 🚚 Gestión de proveedores y compras  
- 🧾 Registro de ventas con comprobantes  
- 🔄 Movimientos de inventario automáticos  
- 🧱 Arquitectura escalable (DAO + Service + Controller + View)

---

## 🧠 Tecnologías utilizadas

- **Python 3**
- **Tkinter** (interfaz gráfica)
- **MySQL** (base de datos)
- Arquitectura **MVC + DAO + Service**
